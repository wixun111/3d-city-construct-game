需要安装的插件
Localization
DOTween
（可选）加载SVG：com.unity.vectorgraphics
如果不用SVG示例里的图标可能无法正常加载，自己替换下。

TMP和Localization可能会有些问题，自己重新配置添加一下内容。
TMP的字体文件不包含在内，同样需要自己配置。

Plugins to install
Localization
DOTween
(Optional) Load SVG: com.unity.vectorgraphics
If you don't use the icons in the SVG example, they may not load properly. Please replace them yourself.

There may be some issues with the TMP and Localization components, so please fix them yourself.
Also, remember to bind the font used by TMP yourself, as the project does not include the font I used.
